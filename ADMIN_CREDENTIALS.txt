Langkah-langkah untuk membuat admin di database:
1. Pastikan database MySQL sudah terinstall dan berjalan
2. Konfigurasi koneksi database di file .env.local
3. Jalankan migrasi database: npx prisma migrate dev
4. Buat admin awal di database: npm run db:seed
